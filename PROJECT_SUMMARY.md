# AirFlow - Project Summary

A modern Next.js academic management system built with TypeScript, Tailwind CSS, and Airtable integration. Features secure NextAuth.js authentication, real-time multi-subject student data aggregation, professional UI with gradient designs, responsive layouts, and dual export functionality (CSV/PDF with visual fidelity). Handles complex student-subject relationships, provides class performance analytics, and includes comprehensive error handling with accessibility support. Production-ready with 20+ assessment fields mapped from Airtable TyMarks table.
